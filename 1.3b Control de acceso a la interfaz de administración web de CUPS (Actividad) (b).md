**Control de acceso a la interfaz de administración web de CUPS**

Por defecto, sólo se puede acceder a la administración web de CUPS desde la máquina local y por parte de usuarios con derecho de administración. El objetivo de esta actividad es conocer las posibilidades de cambiar los permisos de acceso a CUPS tanto de máquinas remotas como usuarios. Para realizar esta tarea se piden las siguientes actividades:

1-Permitir que se pueda acceder a la interfaz web desde cualquier ordenador de la red.

sudo nano /etc/cups/cupsd.conf

Esto permitirá que cualquier ordenador de la red pueda acceder al servidor CUPS.

2-Habilitar la interfaz web sólo en modo local o desde una dirección IP concreta.

Editar el archivo de configuración:

sudo nano /etc/cups/cupsd.conf
