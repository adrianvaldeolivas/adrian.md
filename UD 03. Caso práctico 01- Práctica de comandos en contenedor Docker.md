UD 03. Caso práctico 01- Práctica de comandos en contenedor Docker

Crearemos un contenedor con la imagen base “ubuntu”. Además, al crearlo, dejaremos lista una
“shell” para instalar los programas pertinentes. Esto podemos hacerlo con la orden:
docker run -it --name ejercicio ubuntu /bin/bash
Si todo sale bien nos encontraremos en la “shell” del contenedor, de forma similar a:

<img width="741" height="179" alt="captura 1 aso contenedor" src="https://github.com/user-attachments/assets/a4bda291-ed9a-48c0-a30a-e618e4726777" />

Los alumnos podrán salir de la “shell” con el comando “exit”, lo cual parará el contenedor.
Para volver a acceder, podrán re-arrancarlo con el comando:
docker start -ai ejercicio
