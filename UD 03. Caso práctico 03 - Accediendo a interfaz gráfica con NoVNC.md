UD 03. Caso práctico 03 - Accediendo a interfaz gráfica con NoVNC

1-Ver la version de docker que tenemos:

<img width="496" height="81" alt="captura 1 interfaz grafica" src="https://github.com/user-attachments/assets/33da96de-7cf0-4630-bbf9-e0351f767574" />

2-Para este caso práctico, simplemente creamos un contenedor a partir de la imagen descrita en
https://github.com/theasp/docker-novnc, la cual tiene todo listo para funcionar.
La lanzaremos con el siguiente comando:

<img width="1200" height="647" alt="captura 2 interfaz grafica" src="https://github.com/user-attachments/assets/591899ea-7bfc-4a72-be8c-49270daea1e8" />

3-Ponemos en internet http://localhost:8080/vnc.html

<img width="1288" height="807" alt="captura 3 interfaz grafica" src="https://github.com/user-attachments/assets/e1571b24-0fdb-416c-a375-e918204abc2d" />

4-Abriendo un navegador a la siguiente URL http://localhost:8080/vnc.html se nos cargará un cliente
NoVNC (Cliente VNC en HTML5 y Javascript). Simplemente, conectaremos con dicho cliente y en
nuestro navegador veremos una interfaz gráfica dentro de nuestro contenedor.
