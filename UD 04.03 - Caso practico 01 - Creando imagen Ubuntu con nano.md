UD 04.03 - Caso practico 01 - Creando imagen Ubuntu con nano

1-Crear carpeta para la práctica:

Ve a tu home o Desktop y crea una carpeta:

cd ~
mkdir ubuntu-nano
cd ubuntu-nano

2-Crear el Dockerfile:

Crear el archivo: nano Dockerfile

Pega EXACTAMENTE esto dentro:

# Imagen base ubuntu
FROM ubuntu

# Actualizamos lista de paquetes e instalamos nano
# Las últimas líneas son para hacer la imagen más ligera
RUN apt update && apt install -y nano && apt purge --auto-remove && apt clean && rm -rf /var/lib/apt/lists/*

# Establecemos como comando por defecto de la imagen /bin/bash
CMD /bin/bash

3-Construir la imagen Docker

Dentro de la carpeta donde está el Dockerfile:

sudo docker build -t ubuntunano .
